# 🌟 Registration Form UI

A modern and responsive **Registration Form** built using **HTML & CSS** with a beautiful gradient background and glassmorphism design.

---

## 🚀 Features

* ✨ Clean and modern UI design
* 🎨 Animated gradient background
* 💎 Glassmorphism effect (blur + transparency)
* 🔐 Input focus glow effects
* 📱 Fully responsive layout
* 🖱️ Smooth hover effects on button

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Animations
